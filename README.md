# CESIRA V1 clean package

Questo pacchetto usa i file reali caricati nella conversazione e li dispone
nella struttura corretta per Vite / Vercel.

## Avvio
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy Vercel
- Framework: Vite
- Build command: npm run build
- Output directory: dist
