# CESIRA V1

Struttura pronta per GitHub e Vercel.

## Avvio locale
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy Vercel
- importa il repository su Vercel
- framework: Vite
- build command: `npm run build`
- output directory: `dist`
